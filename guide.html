1软件简介

TradeReplay 是一款 Mac / Windows 原生的「交易复盘训练 + 真实交易日志中心」，覆盖外汇、期货、加密货币、贵金属与股指。

它做两件事：

  • 复盘训练 —— 在历史 K 线上逐根回放，像 MT4/5 的策略测试器一样，一边推进一边下模拟单，练你的盘感、出入场与纪律。
  • 真实交易日志 —— 导入你在 MT4 / MT5 / 期货账户的真实成交，自动算胜率、盈亏比、R 倍数、MAE/MFE，配日历热力图与逐日复盘笔记。

顶部有四个主区，右上角还有两个入口：

  • 概览 仪表盘：训练 / 实盘账户的整体表现、最近成交、行为雷达。
  • 模拟 复盘训练的主战场（图表 + 下单 + 持仓 + 统计）。
  • 实盘 真实成交的复盘（月历 / 交易表 / 统计 / 账户）。
  • 策略 量化策略、指标、交易归集与日志。
  • 数据（右上）行情数据管理，设置（右上）应用设置与许可证。

2下载 · 安装 · 激活

下载 → 安装 → 用激活码解锁 Pro。

  1. 下载：在官网首页下载对应版本 —— Windows 是 .exe 安装包，Mac 是 .dmg。
  2. 安装：Windows 双击 .exe 按提示安装；Mac 打开 .dmg 后把 TradeReplay 拖进「应用程序」。
  3. 激活：打开右上角 设置 → 许可证，输入激活码和购买时使用的邮箱，点「激活」。一个激活码最多可在 2 台设备使用。
还没有激活码？在「许可证」页点 购买 / 续费 即可前往店铺购买，付款后激活码会发到你购买时填的邮箱。换设备时，先在旧设备点「在本设备注销」释放名额。

3准备行情数据

复盘和回测都需要 K 线数据。打开右上角 数据，有三个来源。

系统数据（下载）

内置的外汇 / 贵金属 / 股指 / 加密 / 期货行情。点品种 → 选周期 → 下载即可。中国期货按交易所分类，可选主力连续 / 指数 / 具体合约月份。

自定义（导入 CSV）

导入你自己的数据：OHLC 的 K 线 CSV，或逐笔 Tick CSV（更真实的成交撮合）。填好品种、周期、币种后导入，之后就能用它新建训练。

缓存

当你导入真实成交时，软件会自动下载对应品种的 K 线放在这里，用于把你的真实单子在图上回放。

免费档提供少量主流品种 + 近 1 年历史；Pro 解锁全部品种与完整历史、全部中国期货。

4复盘训练（核心）

在历史行情上逐根推进、下模拟单，这是 TradeReplay 的核心。

4.1 新建一个训练

  1. 进入 模拟，新建一个训练（新建训练）。
  2. 选数据来源（系统 / 自定义）和品种 · 周期。
  3. 设起始时间 —— 想避免「挑顺手的位置」，可以点随机，从数据范围里随机选一个交易日开盘。
  4. 设初始资金（外汇账户为美元，期货账户为人民币，币种由品种自动决定）。
  5. 高级里可调杠杆、点差、手续费、隔夜费等；期货的手续费分开仓 / 平昨 / 平今三笔，保证金率可编辑。
  6. 点创建，训练就会载入图表。

4.2 回放与下单

  • 回放控制：用控制条播放 / 暂停、单步前进 / 后退、加速 / 减速，K 线就会逐根推进。
  • 下单：在下单面板选手数，或切到 风险 $ / 风险 % 让软件按止损距离自动算手数；填止损 / 止盈（可切「点」或「价格」），然后 开多 或 开空。
  • 管理持仓：持仓面板实时显示浮动盈亏，点单子可平仓；「平」按钮可一键平盈利 / 平亏损 / 平多 / 平空 / 平部分。

4.3 复盘后看统计

训练里的统计页会给出资金曲线、回撤、盈亏分布、日历热力图，以及胜率、期望（R 倍数）、最大回撤、MAE/MFE 等指标，帮你看清问题出在出场还是入场。

5真实交易日志

把你在真实账户的成交导进来，自动统计、按天复盘。

5.1 导入真实成交

  1. 进入 实盘 → 账户，点导入。
  2. 选择对账文件：MT4 的 Statement.htm、MT5 的 ReportHistory-*.xlsx，或中国期货的 CFMMC 日结单（.xls，可一次选多份）。
  3. 预览确认后导入 —— 重复的成交（同一票号）会自动跳过，并自动下载对应 K 线以便回放。
所有导入的真实成交与模拟成交分开存放，且只存在本机、不上传任何服务器。

5.2 查看与记录

  • 月历：按天的盈亏热力图，点某天写当日复盘笔记（支持模板与标签）。
  • 交易：全部成交明细表，可加标签、筛选、看每笔的 MAE/MFE。
  • 统计：与复盘训练同一套统计面板，按账户汇总。
  • 账户：余额、出入金、按平台（MT4/MT5/CFMMC）管理多个账户。

6量化策略与回测

把交易想法写成规则，在历史数据上一键回测。

  • 进入 策略 → 量化。内置一批经典策略模板（双均线交叉、MACD、RSI 反转、布林带、海龟突破…），可复制为自定义再改。
  • 规则用 通达信（TDX）公式语法书写 —— 和软件里图表指标是同一套语言，例如做多条件 CROSS(EMA(C,12), EMA(C,26))。可以点插入模板，或让 AI 生成。
  • 写好后点回测：选数据、初始资金、时间范围，运行即可看到成交列表与资金曲线；还能「在数据库查看」，把回测结果生成一个只读复盘。
不熟悉通达信语法也没关系：先从内置模板改，或在量化页用 AI 描述你的想法让它生成规则。

7AI 导师（Mentor）

用你自己的 AI key，给交易做教练式复盘。

  1. 先到 设置 → Mentor AI 填入你自己的 API key（存在本机系统钥匙串里，不会外传，只发给你选的服务商）。
  2. 点右上角 ✨ 打开 Mentor 面板。
  3. 三种能力：复盘（一段时间的总结）、单笔深读（结合行情/时段/波动分析某一笔）、模式发现（跨全部历史找你的行为倾向）。底部还能自由提问。

8设置与快捷键

设置里可调整外观、交易快捷键、AI 与许可证。

  • 个人：昵称、头像（用于「概览」的问候）。
  • 图表：图表时区、默认多图布局。
  • 交易：键盘快捷键（播放/单步、快速买卖、全平等），可自定义。
  • 通用：语言（中 / 英）、明暗主题、检查更新。
  • Mentor AI：填 API key、测试连接。
  • 许可证：激活 / 注销、购买 / 续费。
回放和下单支持键盘快捷键，能大幅提速；具体按键可在 设置 → 交易 查看与修改。

9常见问题

  • 在哪里付费 / 买激活码？
    软件里 设置 → 许可证 → 购买 / 续费，会打开店铺页面。
  • 一个激活码能用几台设备？
    2 台。换设备时先在旧设备点「在本设备注销」。
  • 数据存在哪？安全吗？
    全部存在你本机的私有数据库里，软件不含任何统计/telemetry，不上传你的数据。
  • 必须联网吗？
    只有下载行情、激活、使用 AI 时需要联网；复盘训练本身可离线使用。
  • 怎么更新？
    非 App Store 版在 设置 → 通用 → 检查更新；有新版会提示下载，确认后重启更新。
